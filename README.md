# Node.js (JavaScript) Boilerplate

Node.js (JavaScript) Boilerplate with Gulp and Babel.

## Includes

-   Babel
-   Gulp
    -   `del`
    -   `gulp-image`
    -   `gulp-autoprefixer`
    -   `gulp-sass`
    -   `gulp-csso`
    -   `gulp-bro`
    -   `gulp-typescript`
    -   `babelify`
    -   `gulp-webserver`
    -   `gulp-gh-pages`
